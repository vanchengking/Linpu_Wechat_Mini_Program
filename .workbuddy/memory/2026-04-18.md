## 文化解码页面美化

### 任务
帮用户将文化解码(decode)页面的空白背景更换为更美观的设计，同时保持与其他页面风格一致。

### 完成内容
1. **更新了 decode.wxml**：
   - 将背景图 mode 从 `cover` 改为 `aspectFill`，与其他页面保持一致
   - 添加了背景遮罩层 `bg-overlay`，增强文字可读性

2. **更新了 decode.wxss**：
   - 添加了 `page` 样式，设置背景色为黑色 `#000`
   - 添加背景遮罩层样式，使用渐变遮罩 `linear-gradient(to bottom, rgba(0,0,0,0.1) 0%, rgba(0,0,0,0.2) 50%, rgba(0,0,0,0.4) 100%)`
   - 统一使用 `rpx` 单位（原来是 px）
   - 调整标题样式：白色文字 + 阴影效果
   - 调整竖排文字样式：白色文字 + 双层阴影，字号增大到 72rpx
   - 调整滑动提示位置到底部 120rpx，与其他页面一致

3. **更新了 decode.js**：
   - 调整箭头动画时长从 500ms 到 800ms，节奏更舒缓

4. **创建了 decode.json**：
   - 添加页面配置，设置导航栏为黑色主题

5. **生成了新文件到桌面"林浦"文件夹**：
   - `decode_new.wxml` - 新的页面结构
   - `decode_new.wxss` - 新的样式文件
   - `decode_new.js` - 新的逻辑文件
   - `decode_new.json` - 新的配置文件

### 风格统一要点
- 背景：全屏图片 + 渐变遮罩层
- 文字：白色 + 阴影效果，确保在背景图上清晰可读
- 单位：统一使用 rpx
- 颜色：导航栏黑色，与 scene/guide 等页面保持一致
- 布局：竖排文字居中，错位分布
